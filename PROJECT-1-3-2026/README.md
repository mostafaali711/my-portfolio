# الإخوة للخدمات المنزلية

موقع بسيط لخدمات **السباكة والكهرباء والصيانة المنزلية**.

![صورة الموقع](images/website.png)

## فكرة المشروع

الموقع يقدم خدمات منزلية مثل:

- السباكة
- الكهرباء
- الصيانة المنزلية

يمكن للعميل تصفح الخدمات ومعرفة التفاصيل لكل خدمة.

## صفحات الموقع

- الصفحة الرئيسية
- صفحة خدمات السباكة
- صفحة خدمات الكهرباء
- صفحة الصيانة المنزلية

## التقنيات المستخدمة

- HTML

- متجاوب مع الموبايل
